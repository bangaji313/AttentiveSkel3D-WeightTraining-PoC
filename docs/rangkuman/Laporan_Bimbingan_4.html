📐 Format Tensor Input & Alur Forward Pass
# Format Input Tensor (B=Batch, T=Waktu, L=Landmark, C=Koordinat) input_tensor = shape(16, 64, 33, 3) # (Batch=16, Frame=64, Sendi=33, xyz=3) # Step 1 — Reshape ke format Conv3d x = permute(B, 3, 64, 33).unsqueeze(-1) # → (B, C=3, T=64, L=33, W=1) # Step 2 — Terapkan BSP (Biomechanical Spatial Prior) bsp_weight = torch.sigmoid(biomechanical_spatial_prior) # shape (1,1,1,33,1) x = x * bsp_weight # broadcast → per-sendi weighting # Step 3 — 3× Conv Block ────────────────────────────────────────────────────── x = conv_block_1(x) # → (B, 32, 64, 16, 1) x = conv_block_2(x) # → (B, 64, 32, 8, 1) x = conv_block_3(x) # → (B, 128, 32, 8, 1) # Step 4 — Agregasi Global x = global_avg_pool(x).flatten(start_dim=1) # → (B, 128) # Step 5 — Klasifikasi logits = classifier(x) # → (B, 2) [Benar, Salah]

03 — Mekanisme Biomechanical Attention
final_attentioni  =  raw_attentioni  ×  prior_maski
raw_attentioni = sigmoid(BSPi) — bobot yang dipelajari model dari data
prior_maski = 1.0 (sendi kritis) | 0.1 (sendi lain) — aturan biomekanika
Min-Max Normalization → [0.0, 1.0]  |  Threshold = 0.4 (batas glow merah)
🔴
norm > 0.4 → Sendi Kritis
Glow merah, radius besar (16px), label CYAN terang
⚫
norm ≤ 0.4 → Sendi Non-Kritis
Abu-abu redup, radius kecil (5px), label cyan redup
📊
Top-5 Atensi ditampilkan sebagai HUD overlay pada video dengan nilai raw asli (bukan ternormalisasi)

Temuan Ilmiah Kunci
Model secara tidak disupervisi eksplisit per sendi mempelajari bahwa Lutut (#25: 0.7639) dan Pinggul (#24: 0.7614) adalah sendi paling informatif untuk klasifikasi gerakan benar/salah. Ini konsisten secara biomekanika — kedua sendi tersebut merupakan titik kritis pada Squat depth dan Deadlift hip-hinge, dua gerakan yang paling banyak dalam dataset (171 + 129 = 300 sampel dari 487).

Formula final = raw × prior_mask + Min-Max Normalization berhasil menghasilkan visualisasi yang interpretable: sendi kritis selalu muncul merah terang, sendi non-kritis selalu redup, tanpa ambiguitas. Pendekatan ini siap untuk diintegrasikan ke demo real-time berbasis MediaPipe.
