Laporan Analisis Komprehensif Eksperimen Model AttentiveSkel-3D

A Proof of Concept for Enhancing Weight Training Form Error Detection Using 3D-CNN and Biomechanical Attention Mechanism

Maulana Seno Aji Yudhantara

Presentasi ini merangkum kualitas data, efisiensi arsitektur, bukti empiris studi ablasi, stabilitas pelatihan, kinerja generalisasi, serta interpretabilitas biomekanis model.

Slide 2 | Ringkasan Dataset Akhir

Representasi Data Berbasis Tensor Spasio-Temporal

Dataset tidak menggunakan piksel citra mentah. Setiap video direpresentasikan sebagai tensor matriks spasio-temporal berukuran 64 frame x 33 titik sendi x 3 koordinat.

Total Dataset 487 video repetisi

Slide 3 | Efisiensi Komputasi

Ultra-Lightweight: Beban Parameter Sangat Rendah

Komponen Jumlah Parameter Fungsi Utama
Backbone 3D-CNN 101.858 Ekstraksi fitur spasio-temporal inti
Biomechanical Spatial Prior 33 Prior anatomi berbasis relevansi sendi
Learned Spatial Attention 8.352 Belajar fokus antar-titik sendi secara adaptif
Temporal Attention 129 Penekanan frame penting sepanjang waktu
Total Arsitektur Penuh (Full Model) 110.372 Ukuran file model sekitar 0.43 MB

Kesimpulan: penambahan 3 lapis atensi hanya menambah parameter secara sangat ringan, sehingga efisien dan realistis untuk deployment pada edge device.

Slide 4 | Studi Ablasi

Analisis Kombinasi Atensi pada Validation Set

Model Konfigurasi Val Loss Val Acc Interpretasi
Model 1 Baseline 0.1271 95.89% Titik acuan awal
Model 2 Ablasi A / Tanpa Prior 0.2183 95.89% Memburuk drastis
Model 3 Ablasi B / Tanpa Learned Spatial 0.1581 98.63% Akurasi naik, loss belum optimal
Model 4 Ablasi C / Tanpa Temporal 0.1343 95.89% Stabil, namun tidak terbaik

Sorotan utama: Val Loss terendah pada Model 5 menjadi bukti bahwa penggabungan biomechanical prior, learned spatial attention, dan temporal attention menurunkan tingkat keraguan mesin secara paling efektif.

Slide 5 | Kurva Pembelajaran

Training Dynamics yang Sehat dan Stabil

Kurva Training Loss dan Validation Loss turun beriringan secara sehat. Gap antara akurasi train (100%) dan validation (97.26%) sangat tipis, sehingga tidak menunjukkan gejala overfitting yang berbahaya.

Early Stopping menghentikan proses pelatihan secara optimal di Epoch 38, menjaga efisiensi waktu komputasi sekaligus mempertahankan generalisasi model.

Slide 6 | Evaluasi Kinerja

Performa pada Test Set Unseen Data

Pengujian dilakukan pada 74 data yang tidak pernah dilihat oleh mesin selama pelatihan maupun validasi. Model mencapai performa generalisasi yang sangat kuat.

Akurasi Global 97.30%
F1-Score Gerakan Benar 0.98
F1-Score Gerakan Salah 0.95
Kesalahan Prediksi 2 dari 74

Rincian kesalahan hanya 1 False Positive dan 1 False Negative, menunjukkan keseimbangan deteksi antara dua kelas target.

Slide 7 | Interpretabilitas

Bukti Interpretabilitas sebagai Proof of Concept

Model terbukti dapat memusatkan perhatian pada sendi krusial saat klasifikasi gerakan: contoh pada Squat, bobot atensi tinggi muncul pada Lutut (0.92) dan Pinggul (0.87).

Kesimpulan utama: mekanisme atensi bersifat adaptif terhadap perubahan sudut kamera (Lateral vs Frontal), sehingga perilaku model menyerupai observasi pelatih profesional yang fokus pada titik biomekanis penting.
